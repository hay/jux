# jux
> YAML / Markdown format to write structured narratives

## Install

```
$ npm install --save jux
```

## Usage

```js
var jux = require('jux');

jux.parse('...');
```

Check out the `example.md` and `example.js` files for some guidance.

## API

### jux.parse(input)
